FULL READY SCANNER + DATABASE + WEIGHT PROJECT

Backend API already set:
VITE_API_URL=https://back-a9dq.onrender.com/api

HOW TO RUN:
1. Extract this ZIP
2. Open terminal inside this folder
3. Run:
   npm install
   npm run dev

TEST FLOW:
1. Open in Chrome or Edge
2. Click Connect Devices
3. Select your weighing machine serial port
4. Scan barcode using USB barcode scanner
5. Website fetches bottle data from backend/database
6. Website reads weight
7. Remaining ML is calculated

IMPORTANT:
- Your bottle barcode must already be added in Admin Panel/database.
- Your weighing scale must appear as a serial device.
- Web Serial works only on Chrome/Edge and localhost/HTTPS.
